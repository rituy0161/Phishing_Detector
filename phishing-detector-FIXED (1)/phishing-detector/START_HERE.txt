⚠️ TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Extension won't load in Chrome?
  → Make sure API server is running (keep terminal open)
  → Check for error messages in Ctrl+Shift+J (Chrome DevTools)

No risk badge appears on websites?
  → Wait 2-3 seconds for analysis
  → Refresh the page (F5)
  → Check that API server terminal shows no errors

"Connection Error" in extension popup?
  → API server crashed
  → Check terminal for error message
  → Restart the API server

API server won't start?
  → Make sure Python 3.8+ is installed: python --version
  → Check if another app is using port 5000: netstat -ano | findstr :5000
  → Look at terminal for specific error message
